## 📞 Reporting Vulnerabilities & Auditing

I welcome security audits, code reviews, and vulnerability reports from the community. Since this is an experimental proof of concept, identifying potential exploits is crucial for its development.

If you discover a security vulnerability or privacy leak, please open an Issue or a Pull Request, or contact the maintainer directly.
